Você pode iterar pelo texto e verificar se cada palavra da lista aparece na mesma ordem.

Mas como vou saber se a variável aparece na mesma ordem?

Você pode criar uma variável para armazenar as posições de cada palavra na lista.

Existe uma maneira de facilitar a verificação da ordem?

Você pode usar a função find para localizar a posição de cada palavra no texto e garantir que as posições sejam crescentes. 

posicao = "Este é um exemplo de texto".find("exemplo")
print(posicao)  # Deve imprimir 11, que é o índice onde "exemplo" começa.
