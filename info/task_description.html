Enunciado

Juliana, uma revisora de texto, precisa verificar se uma sequência de palavras aparece na mesma ordem em um texto fornecido. Ela quer uma função que faça essa verificação, considerando que as palavras devem aparecer na mesma ordem e que a verificação é sensível a maiúsculas e minúsculas. Escreva uma função que receba um texto como string e uma lista de palavras, e retorne True se as palavras aparecerem na mesma ordem no texto, e False caso contrário.

Entrada: Uma string (texto) e uma lista de strings (palavras).

Saída: Um valor booleano (True ou False).
